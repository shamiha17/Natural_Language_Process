## Disaster Tweet Classification

An NLP project to classify whether a tweet is about a real disaster or not.

### 📝 Features:
* **Cleaning**: Removed URLs and special characters using Regex.
* **Vectorization**: Used `CountVectorizer` for text-to-numeric conversion.
* **Model**: Trained using **Logistic Regression**.
* **Result**: Achieved **~79% accuracy** on the validation set.

### 🛠️ Tech:
* Python, Pandas, Scikit-learn.
